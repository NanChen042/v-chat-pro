# 仿微信实时聊天系统：前端全栈开发指南 (Vue 3 + Vite + Vant + TS)

本手册提供颗粒度对齐后端的教学方案，涵盖环境搭建、组件封装、WebSocket 未读数管理及 UI 极致优化。

---

## 第一阶段：环境初始化与工具链配置
使用 Vite 搭建 Vue 3 开发环境，配置 Tailwind CSS 4.0 与 Vant 4 自动化引入插件，确保工程化基础稳健。

---

## 第二阶段：底层架构封装

### 2.1 网络与通讯单例
- **请求拦截器 (`request.ts`)**: 统一注入 JWT Token 并对业务异常进行全局 Toast 提示。
- **Socket 增强封装 (`socket.ts`)**: 实现 `currentUserId` 监控。切换账号时强制销毁旧链路并原地重建，彻底解决实时通知漏发问题。

---

## 第三阶段：徽标与未读状态闭环

### 3.1 未读数管理 (Server-Authoritative)
**核心逻辑**: 摒弃前端本地模拟，采用后端驱动。
1. **未读统计**: `ChatService.getSessions` 在聚合会话时，主动统计每个会话中 `isRead: false` 的消息数量。
2. **实时通知**: `App.vue` 监听 `message` 事件。通过 `router.currentRoute` 判断当前视图，仅在非活动对话窗口中增加 `totalUnreadCount` 徽标。
3. **状态消除**: 进入 `ChatView.vue` 时立即调用 `/chat/read` API，并在后端批量更新消息状态，实现“读后即便消”的极致同步。

---

## 第四阶段：社交协作功能 (WeChat 核心)

### 4.1 通讯录与红点系统
- **布局补全**: 在 `ContactView.vue` 使用槽位技术，确保 `van-badge` 徽标完美对齐。
- **导航优化**: 支持从通讯录点击头像直接发起 `/chat/:id` 对话。

### 4.2 自动化打招呼
在好友申请通过瞬间，前端全自动触发 Socket 招呼消息：“我通过了你的朋友验证请求...”，确保会话页 (`IndexView.vue`) 能够立即感知并上屏展示该会话。

---

## 第五阶段：UI 极致细节优化

### 5.1 头像色彩与气泡布局
- **Hex 动态色彩**: 解决 Tailwind 无法渲染后端动态色彩的痛点，采用内联样式 `:style` 映射用户专属 ID 色彩。
- **气泡视觉**: 完美复刻微信绿 (`#95ec69`) 气泡、箭角效果及自动滚动逻辑。

---
本手册为闭环产物，涵盖了从架构到 UI 的所有核心步骤。
